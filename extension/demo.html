Page de démonstration Co-Pilot — Cette page simule une annonce Leboncoin pour tester l'extension.

Comment tester l'extension :

  1. Ouvrir Chrome → chrome://extensions
  2. Activer le Mode développeur (en haut à droite)
  3. Cliquer sur « Charger l'extension non empaquetée »
  4. Sélectionner le dossier extension/ du projet
  5. Lancer le serveur Flask : flask run (ou python wsgi.py)
  6. Cliquer sur le bouton « Tester l'analyse » ci-dessous
